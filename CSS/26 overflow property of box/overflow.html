Hello this is to check the overflow properties of the css and we are gonna experiment on this box by giving it low height and width and applying the css overflow properties to handle the overflow of the text. overflow; auto; is used to give an automatic scroll bar to the box if the the text is overflowing from the box white-space:nowrap; is used to give no wrap to the text written in the box. text-overflow:ellipsis; is used to give 3 dots at the end of the box ending. if the text is greater than the box.
