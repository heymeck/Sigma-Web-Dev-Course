Three ways of linking css in html

  1. Inline CSS:- using style keyword in any tag we can directly design webpage in html.
  2. Internal CSS:- using a style tag in the head and in that style tag we can write our css code.
  3. External CSS:- In this we write our css code in a .css file and then link it with html using link element of html.
